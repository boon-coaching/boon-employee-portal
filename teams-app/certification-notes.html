What This App Does

The Boon Coaching Teams app is an interactive bot (isNotificationOnly: false) that delivers coaching nudges, action item reminders, goal check-ins, and session prep notifications to employees enrolled in a Boon coaching program. Users interact with Adaptive Cards directly in Teams personal chat and can also type messages to the bot.

Bot Testing Instructions

  1. Install the Boon Coaching app from the uploaded manifest package.
  2. Add the bot to a personal chat in Microsoft Teams.
  3. Proactive welcome: On install, the bot sends a Welcome Adaptive Card containing:
    • Welcome message describing the app
    • Feature list: Action Reminders, Goal Check-ins, Session Prep
    • Three buttons: Open Boon Portal, Get Help, Contact Us
  4. Supported text commands (type in chat):
    • hi / hello / hey / help / menu / start → bot responds with the Welcome Adaptive Card
    • Any other text → bot responds with a fallback card pointing the user back to help
  5. Button interactions: Tapping any Action.Submit button on an Adaptive Card returns an updated confirmation card.

Bot Classification

This bot is interactive, not notification-only. isNotificationOnly is set to false in the manifest. All declared functionality (welcome card on install, text command responses, Adaptive Card button handlers) is implemented and validated by testing instructions above.
