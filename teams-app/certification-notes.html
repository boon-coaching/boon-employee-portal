What This App Does

The Boon Coaching Teams app is an interactive bot (isNotificationOnly: false) that delivers coaching nudges, action item reminders, goal check-ins, and session prep notifications to employees enrolled in a Boon coaching program. Users interact with Adaptive Cards directly in Teams personal chat and can also type messages to the bot.

End-to-End Test Flow

Follow Path A (recommended). Path B is only needed if you cannot sideload the app.

Path A — Using your Microsoft test tenant (recommended)

We have side-loaded the Boon Coaching app into the test tenant you provided (M365x34009223.onmicrosoft.com) so the validator user already has the bot available.

  1. Sign in to Microsoft Teams as adelev@M365x34009223.onmicrosoft.com.
  2. Open the Boon Coaching app from the Teams Apps tab and add it to a personal chat.
  3. Expected: the bot proactively sends a Welcome Adaptive Card with three buttons (Open Boon Portal, Get Help, Contact Us).
  4. In a separate browser tab, go to https://my.boon-health.com.
  5. Below the blue "Continue" button, click "Sign in with password instead".
  6. Sign in with the Boon portal credentials shown above.
  7. Click Settings in the left nav. You will see two buttons: Connect Slack and Connect Microsoft Teams.
  8. Click Connect Microsoft Teams. You will be redirected to Microsoft sign-in. Sign in as adelev@M365x34009223.onmicrosoft.com.
  9. Expected: redirect back to portal Settings showing badge "Connected via Microsoft Teams". The Teams bot conversation is now linked to the Boon account.
  10. Return to the Teams chat with the Boon Coaching bot. Type hi, then help, then any other text. Expected: Welcome card for greetings/help, fallback card otherwise.
  11. Wait a few minutes (or click "Send test nudge" via the portal Settings page if available). The bot will send an Adaptive Card with action buttons. Tap any button. Expected: a confirmation Adaptive Card replaces the original.

Path B — Sideload into a different tenant

  1. Download teams-app.zip attached to this submission (or rebuild from the same manifest in the package).
  2. In Teams Admin Center for any tenant where sideloading is enabled, upload the package as a custom app.
  3. From Teams, install the Boon Coaching app to a personal chat for any test user in that tenant.
  4. Continue from step 3 of Path A above.

Bot Behavior — What to Expect

  1. Proactive welcome on install (Adaptive Card): heading, feature list, three buttons (Open Boon Portal, Get Help, Contact Us).
  2. Text commands: hi, hello, hey, help, menu, start, get started all return the Welcome card. No dead ends — every input gets a card response with a forward action.
  3. Fallback: any unrecognized text returns a card with copy explaining the bot's role and a link back to the portal plus a Get Help link.
  4. Action.Submit buttons: tapping any button on a nudge card returns an updated confirmation card and records the response in the user's coaching record.

Bot Classification

This bot is interactive, not notification-only. isNotificationOnly is set to false in the manifest. All declared functionality (welcome card on install, text command responses, Adaptive Card button handlers) is implemented and validated by testing instructions above.
